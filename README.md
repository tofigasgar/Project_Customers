# CUSTOMERS

### The project can be run by following these steps:

1. Clone the repository:
git clone https://github.com/tofigasgar/Project_Customers.git

2. Start the MySQL database using Docker Compose.
3. Run the project.
   Since spring.jpa.hibernate.ddl-auto is set to none, Liquibase will automatically create the necessary tables and stored procedures.
4. Open the Swagger link to view all available API endpoints.
5. Use Postman to test the APIs.


Technologies Used

* Java 21
* Spring Boot 3.5.7
* MYSQL
* Gradle
* Liquibase
* Swagger
* Docker
* Lombok
* Spring Data JPA
* Spring data jdbc
